# Quickstart: Verify Itinerary Refresh

1. Clear browser `localStorage` key `tripiagent-trip-storage` (or use incognito).
2. `npm run dev` → open http://localhost:9001/itinerary
3. Confirm day titles: Jun 30 Gardaland, Jul 1 Monte Baldo, Jul 3 Serravalle.
4. Open Target Bank sidebar — 25+ entries including dining and shopping.
5. Run `npm run lint && npm test && npm run build`.
